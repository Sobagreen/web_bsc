# Telnet Quick Connect (Chrome Extension + Local Bridge)

Это расширение показывает popup, где можно выбрать один из 3 IP и подключиться по Telnet.

## Почему не напрямую из расширения
Браузерные расширения не могут открывать raw Telnet-сокеты (TCP/23) напрямую.
Поэтому используется безопасный и рабочий подход:

- **UI расширения** (popup) в Chrome.
- **Локальный bridge-сервис** (Node.js), который принимает WebSocket от popup и открывает Telnet TCP-соединение.

## Настройка

### 1) Укажите IP и логин/пароль в коде
Файл `popup.js`:

- `TELNET_TARGETS` — 3 IP адреса
- `TELNET_CREDENTIALS.username`
- `TELNET_CREDENTIALS.password`

### 2) Запустите bridge
```bash
cd bridge
npm install
npm start
```

Bridge слушает `ws://127.0.0.1:8765`.

### 3) Установите расширение
1. Откройте `chrome://extensions`
2. Включите **Developer mode**
3. Нажмите **Load unpacked**
4. Выберите папку проекта `/workspace/web_bsc`

## Использование
1. Нажмите на иконку расширения
2. Выберите IP
3. Нажмите **Подключиться**
4. Вводите команды и отправляйте кнопкой **Отправить**

## Важно
- Telnet небезопасен (логин/пароль и трафик без шифрования).
- Рекомендуется использовать это только во внутреннем доверенном контуре.
