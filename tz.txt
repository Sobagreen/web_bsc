Мне нужно веб расширение которое при открытии позволит выбрать один из трех IP адресов и подключится к ним с указанным логином и паролем, который я укажу сам в коде, это подключение по telnet придумай идеально подходящее для этого способ, который позволит делать это прямиком в веб-интерфейсе расширения нашего, обновляй этот файл всегда после внесения изменений

---
Статус выполнения:
1) Создано Chrome-расширение с popup-интерфейсом выбора 1 из 3 IP, кнопками подключения/отключения и отправки команд.
2) Добавлена настройка логина/пароля прямо в коде (`popup.js`).
3) Реализован подход для Telnet через локальный WebSocket->Telnet bridge (`bridge/server.js`), так как прямой Telnet из расширения недоступен.
4) Добавлена инструкция по запуску и установке (`README.md`).
5) Обновлена логика авторизации: логин отправляется только при получении из вывода `ENTER USERNAME`, пароль — только после `ENTER PASSWORD`.
6) Убраны лишние записи в терминал расширения: не добавляются префиксы команд и текст ошибок, в терминале только вывод удалённой telnet-сессии.
7) Отправка команд из UI разрешается только после отправки пароля (после завершения авторизации).
8) Во время подключения в терминале отображается весь входящий текст от bridge/сессии, включая не-JSON сообщения, чтобы не терять вывод до логина.
9) Исправлены причины пустого вывода и обрыва: в bridge исправлена проверка открытого WebSocket (WebSocket.OPEN), добавлен явный `connect-src`/`host_permissions` для подключения popup к `ws://127.0.0.1:8765`, улучшена диагностика закрытия соединения в статусе.
